# Angular first step
### Projeto front-end pessoal preliminar construído para dar mmeus primeiros contatos com o framework Angular.

---

### O website conta com três páginas básicas com funcionalidades próprias. A primeira destas páginas é um hub (home) que centraliza o acesso às outras páginas, a segunda conta com um contador com função de incremento ou decremento, já a terceira é uma página que apresenta uma funcionalidade simplificada de navegaação entre abas dentro de uma mesma rota.
